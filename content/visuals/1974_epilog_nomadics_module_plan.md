# 1974 Whole Earth Epilog / 七、游牧 小册子模块计划

上游源文件：`content/readings/1974_whole_earth_epilog_chapter_translation_zh.md`

本章编辑：Al Perrin。Nomadics 在这里不是浪漫流浪，而是移动能力：找地图、跨边境、骑车修车、住在车里、飞行、航海、露营、攀登、观察野外、理解刀具、鹰猎、捕鱼和车辆改装。

## 模块拆分

1. 移动者的资料入口
   - 锚点：真正的移动者先会找图、读图、查档案。
   - 图像概念：猫在 Explorers Ltd. Source Book、海岸测量档案、地图柜和航拍照片之间拉线。
   - 标签：`Source Book`、`地图`、`航拍`、`档案`、`路线`

2. 旅行、青年旅舍与铁轨
   - 锚点：旅行是进入移动者 grapevine，而不只是买票消费。
   - 图像概念：猫在边境桌、青年旅舍公告板和铁路录音采访之间整理路线卡。
   - 标签：`墨西哥`、`旅舍`、`铁轨`、`路线网`、`习俗`

3. 自行车：低能量交通
   - 锚点：自行车把交通还给身体、注意力和可修理结构。
   - 图像概念：猫在自行车维修架旁调轮圈，旁边有车门风险、车锁、反光背心、旅行路线。
   - 标签：`低能量`、`修车`、`车锁`、`可见性`、`路线`

4. 汽车、车居与摩托
   - 锚点：能修、能临时处理、能居住，才算真正拥有车辆。
   - 图像概念：猫在荒野维修台和可移动车居剖面之间，标出涉水、柴油、VW 手册、可移动家具、摩托临修。
   - 标签：`能修`、`涉水`、`车居`、`VW`、`摩托`

5. 飞行、气象与风险账本
   - 锚点：身体飞行有梦感，但必须把训练不足和死亡数字写进账本。
   - 图像概念：猫在山顶滑翔翼、云图和飞行训练卡之间画风险边界。
   - 标签：`滑翔`、`训练`、`云`、`气象`、`风险`

6. 航海、导航与船
   - 锚点：好的导航教学教语法，不教口诀；船也要能被理解和维护。
   - 图像概念：猫在海图桌上用六分仪、太阳中天、lug sail、船体剖面、独木舟和电气线画关系。
   - 标签：`导航`、`海图`、`帆`、`船体`、`独木舟`

7. 野外、背包、露营与攀登
   - 锚点：野外不是末日幻想，而是冷、湿、累、被咬、攀爬和雪地中下一步如何判断。
   - 图像概念：猫在露营地剖面检查睡袋、背包、攀岩裂缝、雪鞋和救援动作。
   - 标签：`生存`、`背包`、`睡袋`、`攀岩`、`雪鞋`

8. 观察、工具、动物伦理与荒野边界
   - 锚点：真正的户外能力奖励感知和克制，不用过量火力替代判断。
   - 图像概念：猫在野外观察桌旁连接录音机、气枪靶、刀、鹰猎手套、鱼类指南、沙漠测量木桩和 BugVan。
   - 标签：`观察`、`刀`、`鹰猎`、`鱼类`、`荒野`
